Render-time rules:
- Status comes from `help_or_done_msg` when present.
- Completed runs normally show `ALL DONE BABY!` because the CLI stores that
  value in `help_or_done_msg`.
- Otherwise `alldonebaby` displays as complete, `help` displays as help, any
  non-empty manager command displays as iteration, and sparse rows display as
  legacy.
- Compact mode truncates long path, reason, and response fields and includes a
  character-count suffix when text is shortened.
- `--verbose` expands reason, response, and stored-state detail.
- These labels, truncation counts, and summaries are not written back to
  SQLite. The raw columns remain `path`, `cc_response`,
  `ai_decision_output`, `ai_decision_reason`, `help_or_done_msg`, and
  `created_at`.
